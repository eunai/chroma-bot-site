Lexa - Color Bot*

Lexa color bot allows a user to customize their individual role color.

*A new revision of Lexa color bot is in the works to make it multi-purpose

Invite

Features

  • Custom Colors (hex and color names supported)
  • Random Colors
  • Change Prefix (default prefix: %)
  • Simple Permission Manager (limit who can use Lexa)
  • Default Join Role
  • Assign a Designated Channel
  • Discord Support Channel
Command Description
%change [hexcode] Changes to random color. Provide hex color code OR color name for specific color.
%remove Removes your color role, if you have one
%prefix [prefix] Changes this servers command prefix (Admins Only)
%set_channel [channel mention] Set channel to listen to commands in. (set to None to reset)
%set_default_role [role mentions] Set default role(s) to be assigned to new members (Admins Only)
%vote Brings up the vote page. If you enjoy this bot, please don't forget to vote!
%invite Brings up the invitation link.
%servercount Shows how many servers the bot is in.
%help [command] View this help message, or view specific command help

A new and improved revision of the bot is currently in development
